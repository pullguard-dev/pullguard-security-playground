# vulnerable-demo

A **deliberately vulnerable** sample application used to demonstrate
[PullGuard](https://www.pullguard.dev) — security, AI-era, and compliance
scanning on every pull request.

> ⚠️ **This code is intentionally insecure.** It exists only to showcase what
> PullGuard detects. Do not deploy it, copy it, or use any pattern here in real
> software.

Open a pull request and PullGuard scans it automatically — results appear as a
native GitHub Check Run, a PR comment, and an Actions step summary.

See a live example: browse the pull requests in this repository.
